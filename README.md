# setgov.sh
shell script to set cpu frequency governor speed on all cores for debian/ubuntu

1) must has

$ sudo apt-get install linux-tools-common

2) get script

3) to allow only you/user to execute  

$ chmod 700 setgov.sh      

4) open terminal > 

$ /path/to/setgov.sh

or

$ ./setgov.sh #if already in script's directory

freq out!
